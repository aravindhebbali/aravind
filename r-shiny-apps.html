All the R packages authored by me include web applications for interactive analysis built using the shiny package. You can launch these apps directly from the respective packages or can try them live by using the below links. The apps have been deployed on GCP.

olsrr

Build linear regression models using the olsrr package. Quickly iterate through models and generate models with the click of a button.

Launch App
blorr

Build, validate and monitor binary logistic regression models. Includes comprehensive regression output, model fit and validation statistics.

Launch App
descriptr

Generate and visualize descriptive statistics for numeric/categorical data. Explore key statistical distributions using interactive plots.

Launch App


inferr

Beginner friendly interface to perform hypothesis testing using parametric and non-parametric statistical test. Includes flexible input options and structured results.

Launch App
rfm

Tools for customer segmentation using recency, frequency and monetary value analysis. Build customized segments using customer or transaction level data.

Launch App
xplorerr

Apps for data visualization, exploratory data analysis, hypothesis testing, customer segmentation, linear and logistic regression modeling.

Launch App
